# 📁 Folder: `veve/` — Tugas Veve

## 👤 Anggota
**Veve** — Admin Dashboard UI

## 📋 Deskripsi Tugas
Merancang dan mengembangkan tampilan **dashboard admin** yang modern dan responsif, serta halaman dashboard siswa.

## 📂 File di Folder Ini
| File | Deskripsi |
|------|-----------|
| [`index.php`](./index.php) | Dashboard utama admin — statistik, kartu aksi, navigasi |
| [`siswa_dashboard.php`](./siswa_dashboard.php) | Dashboard siswa — lihat profil dan riwayat absensi |

## ✅ Checklist Tugas
- [ ] Dashboard admin dengan tampilan statistik real-time (total siswa, hadir, terlambat)
- [ ] Kartu aksi cepat untuk fitur utama (absensi, registrasi, konsol database)
- [ ] Sidebar navigasi responsif dengan hamburger menu di mobile
- [ ] Toggle tema gelap/terang (Dark Mode / Light Mode)
- [ ] Dashboard siswa dengan tampilan riwayat absensi pribadi

## 🔗 File Terkait (di luar folder)
- `config/koneksi.php` — Koneksi database (koordinasi dengan Aldi)
- `manifest.json` — Konfigurasi PWA
- `sw.js` — Service Worker untuk PWA
